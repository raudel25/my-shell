El shell cuenta con algunas funcionalidades adicionales, como los paréntesis y las comillas que permiten una mayor
interoperabilidad. Tomando en el caso de los paréntesis lo que está contenido en estos como un solo comando modificando
la prioridad de los comandos y en el caso de las comillas se toma lo que está dentro de estas como un solo argumento.
Además de la api proporcionada por la biblioteca readline que permite interactuar con el historial de comandos y el
autocompletado para el sistema de archivos.\
\
Ejemplo:\
\
my_sh $ (command1 && command2) > a.txt  # la salida de <command1> y <command2> se redirige al archivo de texto\
my_sh & echo \"again\"                    # este comando imprime en pantalla again\
\
Implementación:\
\
Se aplica el formato descrito al comando introducido.\
