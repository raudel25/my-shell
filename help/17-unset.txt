El comando unset <c> elimina la variable <c> del sistema con su respectivo valor. Si <c> no existe el comando no tiene
éxito.\
\
Ejemplo:\
\
my_sh $ set c hola\
my_sh $ get c\
hola\
my_sh $ unset c # c se elimina del sistema\
\
Implementación:\
\
Para almacenar las variables se utiliza un array de tamaño fijo y se indexa en el valor de dicha variable por su código ascii.\
